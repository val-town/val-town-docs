## Instructions

1. Install the [Val Town CLI](https://github.com/val-town/vt) and log in
2. Install [OpenAI Codex](https://developers.openai.com/codex/cli) and log in
3. Copy-paste our [system prompt](https://www.val.town/x/valdottown/Townie/code/prompts/system_prompt.txt) into `SYSTEM_PROMPT.md` at the root of your local val directory
