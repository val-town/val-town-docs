## Instructions

1. Install the [Val Town CLI](https://github.com/val-town/vt) and log in
2. Install [Warp](https://warp.dev)
3. Copy-paste our [system prompt](https://www.val.town/x/valdottown/Townie/code/prompts/system_prompt.txt) into [`WARP.md`](https://docs.warp.dev/knowledge-and-collaboration/rules)
