## Instructions

1. Install the [Val Town CLI](https://github.com/val-town/vt) and log in
2. Open [Cursor](https://cursor.com)
3. Copy-paste our [system prompt](https://www.val.town/x/valdottown/Townie/code/prompts/system_prompt.txt) into [`.cursor/rules`](https://docs.cursor.com/context/rules-for-ai)
