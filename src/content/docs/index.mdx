Val Town lets you (or your agent) write and run JavaScript, automatically deployed to a live URL in \<100ms. Create APIs, crons, agents, static sites, apps, email handlers, and more in your browser (or from your LLM). For example, use Val Town in Claude Code with our [MCP server](/guides/prompting/mcp/):

<Code
  code={`claude mcp add --transport http val-town https://api.val.town/v3/mcp`}
  lang="bash"
/>

If you're new here and want to learn fundamentals the old fashioned way, read: [What is a val?](/vals/)
